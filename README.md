# Snake Prototype - Sayel Game

## Développé par Sayel

Bienvenue sur le prototype du Sayel Game !  
Le serpent s'appelle **YellowSnake**. Ce jeu est un Snake moderne jouable sur **desktop, tablette et mobile**, avec des murs animés comme obstacles et des œufs piégés pour un challenge supplémentaire.

---

## Description du jeu

- Snake classique : grandit en mangeant des œufs.
- Mort si touche son corps ou un mur.
- Œufs normaux : augmentent taille et score.
- Œufs piégés : rouges fissurés, clignotent, KO instantané.
- Murs animés subtilement pour un rendu réaliste.
- Trois niveaux de difficulté : Easy, Challenge, Hardcore.

---

## Instructions pour jouer

- **Desktop / Laptop** : flèches clavier ↑ ↓ ← → pour déplacer YellowSnake.
- **Mobile / Tablette** : 
  - Swipe tactile pour changer de direction,  
  - ou utilisez les boutons directionnels à l’écran si activés.
- Objectif : manger les œufs et éviter les murs et œufs piégés.
- Timer visible en haut pour suivre la durée du niveau.
